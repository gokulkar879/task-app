
# Task App

A simple task management app to let autheticated users create and delete a task.

# Features: 
1. Authentication
2. clean UI
3. Drag and Drop
4. Assign Severity level to each task.
5. Add a timer to each task.
6. Delete a task

# How to use the code.
1. Clone the repository
2. Run npm install
